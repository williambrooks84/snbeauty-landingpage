# Snbeaty landing page

## Installation

Installer les dépendances:

```bash
# pnpm
pnpm install

```

##Développement

Démarrer le développement sur `http://localhost:3000`:

```bash
# pnpm
pnpm dev
```

## Production

Compiler le projet pour déploiement:

```bash
# pnpm
pnpm build
```

Visualiser la version compilée en local:

```bash
# pnpm
pnpm preview

```
